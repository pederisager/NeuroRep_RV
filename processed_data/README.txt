#####################
Content descriptions.
#####################

- dataset_A: 
	A large sparse dataset containing 1500 records chosen from a filtered list of 2741 WoS records. Sample size for all studies included in this set will be hand coded. The scripts used to filter the WoS data and generate dataset A can befound in the analysis/ directory (wos_data*.R scripts). 

- metadata_WoS/: 
	This folder contains processed data extracted from the downloaded WoS data in raw_data/, as well as a hand-curated list of to-be-excluded article keywords, used for filtering the data before extracting dataset A.
