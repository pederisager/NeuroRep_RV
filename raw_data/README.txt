Content descriptions.

- pilot_sample_study_data.tsv: 
	This file contains the meta-data of a pilot sample of 103 studies that are topically relevant to this replication project. 
	The studies were curated partially from the journal "Social Cognitive and Affective Neuroscience" Vol. 11, issue 1-3, 
	and partially from top search results for "fmri AND social" in ScienceDirect, filtering on article type "Research articles". 

- source_data: 
	This folder contains meta-data for the individual DOIs included as well as miscelaneous data files related to the curation process.
