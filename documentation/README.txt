- RV_coding_instructions
	This document specifies what data should be coded (sample size etc.) and the details of how they should be coded (e.g. only code sample size for participants that underwent fMRI) for data that needs to be coded manually. 

- RV_data_extraction_guidelines
	This document provides step-by-step guidelines for how to extract the automatically extracted data that was used in this project (bibliometrics, PDF files, etc.).
